12 Week Year App

App web mobile-first para implementar o método “1 Ano em 12 Semanas”: planeje ciclos de 12 semanas, execute com foco semanal (WAM), mensure leadings e laggings e feche cada semana com pontuação e retrospectiva.

Stack sugerida: Next.js + Tailwind + Supabase (Auth/Postgres/RLS) + Recharts + n8n (webhooks).
Deploy: Vercel (front) + Supabase Cloud (DB/Storage/Auth).
